1. 싱글톤이란 무엇인가?
특정 클래스를 미리 독립적인 객체로 분류하여 메모리에 할당한다.(정적할당) 그로인해 해당 클래스로 객체를 만드는건 불가능해지기 때문에 단 하나만 존재하게 된다.

2. 싱글톤을 만들기 위해서 사용되는 기법들과 사용 이유를 서술하라.
- 동적 할당 객체 : 싱글톤으로 생성한 객체를 필요한 순간에 사용하고 사용이 끝나면 지우기 위해서이다.
- 정적 멤버 변수 : 싱글톤은 유일한 하나의 객체 선언을 의미하기 때문에 정적으로 미리 메모리를 할당받기 위해서이다.

3. 싱글톤 패턴의 응용 게임 제작에서 예를 찾아보자.
- 대표적으로 GameManager가 있으며 private static GameManager instnace = null; 과 같이 선언한다. 이후 똑같이 해당 instance가 null일 경우 할당한다. 사용 시에는 instance.GameOver()와 같이 쉬운 접근이 가능하게 된다.
